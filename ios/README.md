# Picassa

Picassa je iOS aplikácia na správu udalostí a kalendára, vytvorená pomocou SwiftUI a The Composable Architecture (TCA).

## Funkcie

- 📅 Kalendárový pohľad s mesačným zobrazením
- ➕ Vytváranie a úprava udalostí
- 👥 Správa účastníkov udalostí
- 🔍 Detailný pohľad udalostí
- 🌍 Lokalizácia (SK)
- 🔥 Firebase integrácia pre ukladanie dát

## Technológie

- SwiftUI
- [The Composable Architecture (TCA)](https://github.com/pointfreeco/swift-composable-architecture)
- Firebase/Firestore
- Swift Package Manager

## Požiadavky

- iOS 16.0+
- Xcode 14.0+
- Swift 5.7+
